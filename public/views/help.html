Help Page

Hi team! This is a simple Self Service Application where you can run regression suites for SAP functionalities. The following steps will help you in understanding the process:

  1. Enter ther username in the respective input field.
  2. Select Scenarios which you want to run by selecting from the dropdowm menu available.
  3. Upload the respective testdata file for the scenario which you want to run.
  4. You also can run multiple scenarios by adding rows and following step 3.
  5. Once you have finished uploading your testdata files click on execute script.
    Note: Dont navigate or refresh until the testdata files are uploaded to the server.
  6. You can click on Get Status to check your result status.
    Note: Once you navigate to the next page your data will be in progress and the home page will be refreshed.
  7. Once the script execution has been completed the results will be mailed to you.
  8. If you have any queries mail to us at wpsa-qa@juniper.net. Thank you!
